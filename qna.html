Question-1: What are the different ways to select an element in the DOM?

We can select an element in the DOM in 5 ways:

  1. document.getElementById("id-name")
  2. document.getElementsByClassName("class-name")
  3. document.getElementsByTagName("tag")
  4. document.querySelector("selector")
  5. document.querySelectorAll("selector")

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Difference between innerHTML, innerText, and textContent:

  1. innerHTML: We can get the inside content of an element & also can set new HTML content inside the element.
  2. innerText: We can get or set only the visible texts inside an element.
  3. textContent: We can get and set all text even if the element is hidden.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a parent listens on its child elements by using a single event listener. This is done by taking help of event bubbling.

Question-4: What is event bubbling in the DOM?

When an event on a child element bubbles up to its parent element, we call it Event Bubbling.

Question-5: How do you create, add, and remove elements using JavaScript?

How we create, add, and remove elements using JS:

  1. Creating an element: let myNewDiv = document.createElement("div");
  2. Adding it to DOM: document.body.appentChild(myNewDiv);
  3. Removing an element: myNewDiv.remove();
